# Część 3. Nowy projekt
Stwórz nowy projekt. Najlepiej zrobisz, jeśli projekty będiesz trzymać w folderze Dokumenty. Najłatwiej będzie je wtedy znaleźć.

## Otwórz folder Dokumenty
Zacznij od otwarcia folderu Dokumenty w Visual Studio Code. Są na to dwa sposoby:  
1. Otwórz folder Dokumenty na dysku, w eksploratorze plików. Naciśnij prawym przyciskiem myszy na tło folderu i wybierz opcję "Open with Code".
2. Otwórz Visual Studio Code. W zakładce "**File**" wybierz opcję "**Open Folder...**" i wybierz folder Dokumenty.
(Skrót klawiszowy: Ctrl+K -> Ctrl+O)

## Stwórz nowy projekt
Istnieją dwa sposoby stworzenia nowego projektu w C#, z wykorzystaniem interfejsu graficznego, lub przez konsolę. 
### Interfejs graficzny
Kroki tworzenia nowego projektu:
File → New File → .NET: New Project… C# Dev Kit → Console App. → Name LekcjaX → Folder Documents → Show all template options → Do-not-use-top-level-statement = true→ Create project.

### Konsola
Otwórz konsolę za pomocą skrótu klawiszowego CTRL + \` (tylda znajduje się w lewym górnym rogu klawiatury, pod klawiszem ESC)
W konsoli wpisz `dotnet new console -n LekcjaX --use-program-main`. Zastąp X numerem dzisiejszej lekcji.
Aby przejść do folderu z projektem wpisz w konsoli `cd LekcjaX`

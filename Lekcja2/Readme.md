# Część 3. Nowy projekt
Stwórz nowy projekt. Najlepiej zrobisz, jeśli projekty będiesz trzymać w folderze Dokumenty. Najłatwiej będzie je wtedy znaleźć.

## Otwórz folder Dokumenty
Aby stworzyć nowy projekt zacznij od otwarcia folderu Dokumenty w Visual Studio Code. Są na to dwa sposoby:  
1. Otwórz folder Dokumenty na dysku, w eksploratorze plików. Naciśnij prawym przyciskiem myszy na tło folderu i wybierz opcję "Open with Code".
2. Otwórz Visual Studio Code. W zakładce "**File**" wybierz opcję "**Open Folder...**" i wybierz folder Dokumenty. (Skrót klawiszowy: Ctrl+K -> Ctrl+O
